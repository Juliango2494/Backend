Gestion de biblioteca

Por: Julian Gordillo

Bienvenido a la app de gestión básica de biblioteca, utilizando Typescript en el backend para simular la gestion de datos en una biblioteca con las opciones de "Agregar", "Eliminar", "Mostrar" y "Buscar". Selecciona la opcion deseada en el siguiente Menu:



Agregar un nuevo libro

Buscar un libro

Mostrar todos los libros

Eliminar un libro
